# Contributing

Thank you for your interest in contributing!

## Quick Start

```bash
# Install uv if needed
curl -LsSf https://astral.sh/uv/install.sh | sh

# Setup
uv sync --all-extras

# Install poe globally (optional)
uv tool install poethepoet
```

## Available Commands

View all available tasks:
```bash
poe --help
```
